PYTHON

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics. It is both useful for Rapid Application Development and as a scripting language. Programmers prefer Python because of the increased productivity it provides. Python is a popular choice for developing Machine Learning applications.
This Course attempts in equipping you with required programming skills using Python, as any data science and machine learning implementation asks for adequate knowledge on python to solve various use cases.




Course duration: 25 hours Classroom session + 40 hours of lab access with assistance


Lunch and evening tea will be served

Course delivery: Classroom session on concepts supported by in-depth hands-on sessions with 1 instructor for every 3 students

The course contents include:
 
- Installing and getting started with python on windows environment, meeting the IDE: Jupyter required to work with Python Interpreter for easy development.
- Introduction to the little containers: Variables in Python to perform operations and store the results
- Understand basic arithmetic in Python and get started with our first project work of the course with some algebraic coding challenge
- Learn to use strings and texts in Python and get the new project on strings completed
- Logical Computation and Conditional Flow: Understand different Logical and Conditional Py operators.
- Data Structures in Python: The best methods of structuring data to implement a logic using Lists, Tuples, Sets and Dictionaries and verify the learnings by solving realistic use cases and projects with the help of Py Data Structures
- Loops : Going around different loop techniques and implement projects like conversation simulator and Translator to gain mastery over the section.
- Functionize the code: Make code reusable, readable and flexible using functions in python and understanding the scope of it in a program. This is implemented on the Project
- Code with objects and classes: Learn object oriented programming and use it as a project using Python
- Expressions in Python: Here you will different available expressions in python
