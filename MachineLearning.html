MACHINE LEARNING AND DEEP LEARNING

Machine learning is a type of artificial intelligence (AI) that enables computer applications to predict outcomes without being explicitly programmed. It is based on the idea that Machines should be able to learn and adapt through experience. Machine learning algorithms are often categorized as being supervised or unsupervised.

Course duration: 35 hours Classroom session + 40 hours of lab access with assistance

Lunch and evening tea will be served

Course delivery: Classroom session on concepts supported by in-depth hands-on sessions with 1 instructor for every 3 students

The course includes:
- Introduction to Machine Learning. why Machine Learning and Why Now?
- Statistical Modeling and Forecasting Approach in Machine Learning
- Instance based and Rule based approach towards Supervised in Machine Learning
- Support Vector Machine and Kernel Tricks in SVM in ML using Python
- Real time implementation of Unsupervised Learning with Python
- Dimensionality Reduction and Model Selection
